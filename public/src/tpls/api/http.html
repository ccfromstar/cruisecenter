$http

angular提供了$http服务来同服务端进行通信

1 $http.get GET方法

$http.get('/someUrl').
success(function(data, status, headers, config) {
  console.log(data);
}).
error(function(data, status, headers, config) {
  console.log("error");
});

 

2 $http.post POST方法

$http.post('/someUrl', {parm1:'hello'}).
success(function(data, status, headers, config) {
  console.log(data);
}).
error(function(data, status, headers, config) {
  console.log("error");
});
