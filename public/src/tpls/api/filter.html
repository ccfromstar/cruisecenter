$filter

Filter是用来格式化数据用的。

一.AngularJS内建的Filter

1.1 currency 格式化货币

将12格式化为货币，默认单位符号为 '$', 小数默认2位

{{ 12 | currency }}
执行结果：{{ 12 | currency}}

将12.45格式化为货币，使用自定义单位符号为 '￥', 小数默认2位

{{ noanalysis_left}} 12.45 | currency:'￥' }}
执行结果：{{ 12.45 | currency:'￥'}}

将12.45格式化为货币，使用自定义单位符号为 'CHY￥', 小数指定1位, 会执行四舍五入操作

{{ noanalysis_left}} 12.45 | currency:'CHY￥':1 }}
这里小数点后保留1位没起作用,可能是AngularJS版本的问题
执行结果：{{ 12.45 | currency:'CHY￥':1}}

将12.55格式化为货币， 不改变单位符号， 小数部分将四舍五入

{{ noanalysis_left}} 12.55 | currency:undefined:0 }}
执行结果：{{ 12.55 | currency:undefined:0}}

 

1.2 date 格式化日期

{{ noanalysis_left}} 1304375948024 | date:"yyyy-MM-dd hh:mm:ss" }}
执行结果：{{ 1304375948024 | date:"yyyy-MM-dd hh:mm:ss"}}

 

1.3 number 格式化数字

{{ noanalysis_left}} 1.234567 | number:1 }}
执行结果：{{ 1.234567 | number:1}} 
{{ noanalysis_left}} 1234567 | number }}
执行结果：{{ 1234567 | number}} 
{{ noanalysis_left}} true | number }}
执行结果：{{ true | number}}

 

1.4 uppercase，lowercase 大小写转换

{{ noanalysis_left}} "lower cap string" | uppercase }}
执行结果：{{ "lower cap string" | uppercase}} 
{{ noanalysis_left}} "TANK is GOOD" | lowercase }}
执行结果：{{ "TANK is GOOD" | lowercase}}

 

1.5 json

将对象格式化成标准的JSON格式

{{ {name:'Jack', age: 21} | json }}
执行结果：{{ {name:'Jack', age: 21} | json }}

 

二.自定义Filter

angular.module('tanktest', []).filter('tankreplace', function() {  
  return function(input) {  
    return input.replace(/tank/, "str1")  
  };  
});
{{ noanalysis_left}} "tank is GOOD" | tankreplace }}
执行结果：{{ "tank is GOOD" | tankreplace}}

 

三.Filter可以被链式使用

{{ expression | filter1 | filter2 | ... }}

{{ "TANK is GOOD" | uppercase | tankreplace }}
执行结果：{{ "TANK is GOOD" | lowercase | tankreplace }}
